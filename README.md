## 📝 概要
「死ぬまでにやりたい事」に焦点を当てた、【todoアプリ】です。

ユーザーは、「やりたい事」とその達成期限などを自由に設定できます。

このアプリには、「やりたい事」を達成する為の行動を促す機能が、いくつかあります。
